I'll run through each of these quickly

Which will hopefully show why learning R would be useful

open source

  • it's free to use on Windows, Linux and OSX

  • source code is open to read, inspect, learn from

  • many eyes means improvements/bugs can be made/fixed

  • easy to collaborate as everyone should have access to the same tools

  • ability to extend the language and build packages yourself

  • plenty of useful tutorials/help pages to learn

if you've a question about R, there's a good chance someone has already asked the very same, or similar, and someone else has already answered

programming language

  • language designed specifically for data analysis

  • powerful and flexible

  • used interactively or write scripts to automate tasks

designed by statisticians for statistics, so can appear a little idiosyncratic if coming to R having used other languages

the first language I made an effort to learn, so didn't have any expectations

I'll share some examples of using scripts to automate tasks later

Hadley Wickham's impact on R is seismic

packages built by Hadley Wickham (chief scientist at RStudio) make up the hadleyverse

they range from visualisation packages like ggplot2 and ggvis, to data manipulation packages like dplyr and plyr

also made packages to make the development of packages easier, impact he's had is seismic

if you use R for any period of time, you will enter hadleyverse

a community

  • dedicated and passionate community who contribute to improving R

  • over 7000 R packages on CRAN

  • packages built by experts in their field, including
